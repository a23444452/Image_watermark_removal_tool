# 快速開始指南

## 5 分鐘上手指南

### 1. 安裝 (1 分鐘)

```bash
cd watermark-remover
pip install -e .
```

或手動安裝：
```bash
pip install opencv-python numpy Pillow PySide6
```

### 2. 啟動程式 (10 秒)

```bash
python run.py
```

### 3. 使用程式 (3 分鐘)

#### 步驟 1: 開啟圖像
點擊 **「開啟圖像」** 按鈕，選擇要處理的圖像檔案

#### 步驟 2: 選擇浮水印
兩種方式擇一：

**方式 A: 手動選擇（推薦）**
- 在圖像上按住滑鼠左鍵並拖曳
- 框選包含浮水印的區域
- 會出現紅色矩形框

**方式 B: 自動偵測**
- 點擊 **「自動偵測浮水印」** 按鈕
- 系統會自動在四個角落尋找浮水印

#### 步驟 3: 移除浮水印
點擊 **「移除浮水印」** 按鈕

#### 步驟 4: 儲存結果
點擊 **「儲存圖像」** 按鈕，選擇儲存位置

#### 步驟 5: 如需重試
點擊 **「重置」** 按鈕恢復原始圖像

## 提示與技巧

### ✓ 最佳實踐
- 使用高解析度原圖
- 精確框選浮水印區域
- 儲存為 PNG 格式保持品質
- 處理前先備份原始檔案

### ✓ 適合處理的浮水印
- 角落的小型文字浮水印 ✓
- 單色或半透明標誌 ✓
- AI 生成圖像的浮水印 ✓
- 邊緣的簡單標記 ✓

### ⚠ 可能效果不佳
- 大面積複雜浮水印 ✗
- 重複圖案浮水印 ✗
- 複雜背景上的浮水印 ✗

## 常見問題

**Q: 自動偵測找不到浮水印？**
A: 改用手動選擇。自動偵測主要針對四個角落的文字浮水印。

**Q: 處理後有殘影？**
A: 重新選擇更精確的區域，或稍微擴大選擇範圍。

**Q: 可以處理影片嗎？**
A: 目前僅支援靜態圖像。
